# MONOgement
## 製品概要
### MONO Tech

### 背景（製品開発のきっかけ、課題等）
研究室でホワイトボードやサーバなど，今使用中かを知りたい場面が多くあった．  
どんなものでも多人数で共用するものに対して使用状況を知りたい．
### 製品説明（具体的な製品の説明）
どんなものでもQRコードを貼り付けることで物品状況を管理できる．  
情報としては誰がいつまで使用中か，誰がどの期間予約しているかがわかる．  
PC，Android，iOSから予約や情報閲覧を行うことができる．
